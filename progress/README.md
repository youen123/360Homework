### progress

#### 调用
new Progress(type, percent, el)

API
--------------------------------------------------
Property | Description | Type | Default  
type | 类型 'bar' | 'circle' | 'sandGlass' | --  
percent | 进度百分比 0 - 100 | number | 0  
setType | 设置类型 | function(type) | --  
setPercent | 设置进度 | function(percent) | --  
getPercent | 获取进度 | function() | --
